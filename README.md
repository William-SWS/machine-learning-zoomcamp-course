Este repositório foi criado para documentar minha jornada no curso **Machine Learning Zoomcamp**, organizado pela comunidade DataTalks.Club.

## 🎯 Objetivo

O objetivo deste repositório é:

* Consolidar o aprendizado em **Machine Learning**
* Organizar exercícios, anotações e projetos desenvolvidos ao longo do curso
* Servir como portfólio prático para futuras oportunidades acadêmicas e profissionais
* Explorar conceitos fundamentais e avançados da área

---

## 🧠 Sobre o curso

O **Machine Learning Zoomcamp** é um curso intensivo focado em aplicações práticas de aprendizado de máquina. Ele cobre todo o pipeline de ML, incluindo:

* Preparação e exploração de dados
* Modelos de classificação e regressão
* Validação e métricas de avaliação
* Deploy de modelos
* Ferramentas do ecossistema de ML
